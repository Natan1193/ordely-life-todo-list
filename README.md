# ✅ Orderly Life To-Do List

Uma aplicação de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro.

O projeto foi criado com foco em prática de manipulação de DOM, responsividade, armazenamento local e organização de código front-end.

---

## 🚀 Funcionalidades

- Adicionar tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Persistência de dados com localStorage
- Layout responsivo
- Interface moderna e personalizada

---

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage API

---

## 📱 Responsividade

O projeto foi desenvolvido utilizando abordagem mobile-first e técnicas modernas como:

- Flexbox
- Media Queries
- clamp()
- Variáveis CSS
